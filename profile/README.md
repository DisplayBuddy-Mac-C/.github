## 📋 Overview

<a href="#displayBuddy">DisplayBuddy</a> is a focused external monitor control utility for Mac — using the DDC (Display Data Channel) protocol to control monitor hardware settings directly from the Mac menu bar, making brightness, volume, and input source adjustments as quick and accessible as adjusting the built-in display.

The <a href="#displayBuddy">DDC hardware brightness control</a> sends brightness adjustment commands directly to the external monitor's hardware through the DDC protocol — the same protocol that allows monitors to communicate configuration with connected devices. Rather than controlling a software overlay that simulates brightness change (which doesn't actually reduce the monitor's light output), DDC control adjusts the monitor's actual hardware backlight brightness as if the physical OSD brightness buttons were used. The result is genuine hardware brightness reduction that reduces eye strain and power consumption, not a software dimming layer. The <a href="#displayBuddy">menu bar quick access</a> places the monitor controls in a compact menu bar panel — clicking the menu bar icon reveals sliders for brightness and volume adjustment, input source selection, and other configured controls, all accessible without leaving the current application or touching the monitor's physical buttons.

The <a href="#displayBuddy">keyboard shortcut brightness control</a> configures keyboard shortcuts for incremental brightness adjustment — raising or lowering external monitor brightness with a key press the same way macOS's brightness keys control the built-in display. This keyboard control makes brightness adjustment instantaneous and keeps hands on the keyboard without interrupting workflow to reach for monitor buttons or open a menu. The <a href="#displayBuddy">input source switching</a> changes the monitor's active input from the Mac — switching between connected sources without using the monitor's physical OSD menu, particularly useful for monitors connected to multiple devices that require frequent input changes.

<a href="#displayBuddy">Multiple monitor management</a> controls all connected external monitors individually — separate brightness settings for each monitor, independent input source selection, and per-monitor control from the same menu bar panel. <a href="#displayBuddy">Volume control for monitors with built-in speakers</a> adjusts the monitor's hardware speaker volume through DDC — useful for monitors whose built-in speakers are the audio output for a Mac setup.

<a href="#displayBuddy">Display information panel</a> shows current monitor settings — the active brightness level, current input source, monitor model information, and other DDC-readable display parameters. <a href="#displayBuddy">Setapp subscription availability</a> includes DisplayBuddy for subscribers who want monitor control alongside other Mac utilities through a single subscription.

## ❓ FAQ

<details>
<summary>Does DisplayBuddy control external monitor brightness through software overlay?</summary>

No. DisplayBuddy uses the DDC protocol to adjust the monitor's actual hardware backlight brightness — genuine hardware control rather than a software dimming layer.

</details>

<details>
<summary>What is DDC?</summary>

DDC (Display Data Channel) is a standard protocol for communication between displays and computers. It allows software to read and write display settings including brightness, volume, and input source.

</details>

<details>
<summary>Does DisplayBuddy work with all external monitors?</summary>

DDC control requires monitors and connections that support the DDC protocol. Most modern monitors on HDMI, DisplayPort, and USB-C connections support DDC. Specific compatibility depends on the monitor model and cable. Some USB hubs and adapters can interfere with DDC communication.

</details>

<details>
<summary>Can DisplayBuddy switch monitor input sources?</summary>

Yes. Input source switching changes the monitor's active input through DDC without using the physical OSD menu.

</details>

<details>
<summary>Does DisplayBuddy support multiple monitors?</summary>

Yes. Multiple connected external monitors control individually with separate settings for each display.

</details>

<details>
<summary>Does DisplayBuddy have keyboard shortcuts?</summary>

Yes. Configurable keyboard shortcuts adjust external monitor brightness incrementally — the same experience as the Mac's built-in brightness keys.

</details>

<details>
<summary>Is DisplayBuddy available through Setapp?</summary>

Yes. DisplayBuddy is available through the Setapp subscription service.

</details>

<details>
<summary>Does DisplayBuddy control monitor speaker volume?</summary>

Yes. Monitors with built-in speakers that support DDC volume control can adjust volume from DisplayBuddy.

</details>

<details>
<summary>Does DisplayBuddy run on Apple Silicon?</summary>

Yes. Current versions support Apple Silicon Macs natively.

</details>
